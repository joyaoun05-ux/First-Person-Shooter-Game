
 Wave Survival Shooter

 Game Overview
A first-person wave-based survival shooter where the player fights against increasingly difficult waves of enemies. The goal is to survive as long as possible while managing resources, upgrading abilities, and adapting to different enemy types.



 Controls
- **WASD** – Move
- **Mouse** – Look
- **Left Click** – Shoot
- **R** – Reload
- **E** – Interact (shop / perks / mystery box)
- **Shift** – Sprint
- **Space** – Jump



 Enemy Types
- **Normal Enemy** – Basic enemy with balanced stats  
- **Healer Enemy** – Heals nearby enemies  
- **Tank Enemy** – High health, slow movement  
- **Fast Enemy** – Very fast but low health  
- **Shield Enemy** – Takes reduced damage from the front  

---

 Systems & Features
- Wave-based enemy spawning with increasing difficulty  
- Economy system (earn money from kills)  
- Mystery box with random rewards  
- Health recovery system  
- Perks / buffs to improve player performance  
- Hit markers and damage feedback  
- Wave transition screen with pause  
- Main menu and game over system with high score tracking

 Level Design
- Two distinct zones:
  - Main combat arena  
  - Shooting range area  
- Environmental cover:
  - Containers  
  - Sandbags  
  - Concrete blocks  
- Multiple enemy spawn points to prevent camping
